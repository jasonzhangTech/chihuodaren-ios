# 吃货达人 iOS

吃货达人是一款 iPhone 原生美食日志 App。当前版本聚焦单机可用：快速记录探店照片、店名、类型、地址、评分和推荐菜，并在之后用本地日志辅助决定“今天吃啥”。

## 当前状态

- 目标平台：iPhone，iOS 17 及以上
- 技术栈：SwiftUI、SwiftData、本地优先
- Bundle ID：`com.jasonzhangtech.chihuodaren`
- 数据策略：本地持久化，后续更新默认保留已有数据
- 社区能力：第一版不做社区

## 已实现功能

- 日志首页：本地美食日志流、搜索、动态筛选、年月日时间展示
- 动态筛选：筛选项来自已有日志的类型，默认包含 `全部`、`踩雷`
- 新建探店：店名必填，草稿自动保存，避免记录失败丢数据
- 历史日期：新建和编辑日志可用 iOS 原生日期选择器选择探店日期，默认今天
- 类型选择：火锅、烧烤、烤肉、正餐、小吃、咖啡、甜品、粉面、其他
- 地址记录：通过地图搜索和点选保存店铺地址，进入地址选择时主动获取当前位置，并支持详情页导航
- 距离展示：首页和详情页显示店铺距离当前位置的距离
- 照片记录：支持相机拍照和相册选择，多图使用九宫格样式，首页封面最多展示 6 张
- 默认封面：日志没有照片时，首页卡片展示默认封面，不显示添加照片提示
- 手动评分：环境、服务、菜品三项评分，支持 0.5 星，最终评分取平均值
- 评价标记：推荐 / 踩雷，大拇指样式，推荐使用主题暖色
- 推荐菜：可选输入，用顿号分隔
- 详情页：可滑动查看完整照片、评分、地址和导航入口
- 吃啥页：使用和日志首页一致的类型筛选逻辑，从本地非踩雷日志中推荐记录
- 应用图标：已配置 AppIcon 资源

## 已移除或暂不包含

- 不再接入或模拟大众点评 / 高德评分与推荐菜自动补全
- 不再使用店名自动识别类型
- 不再保留一句话口述、备忘、下次还去吗
- 不再做默认位置保护文案，改为展示距离当前位置
- 不包含社区、账号体系和云同步

## 运行方式

用 Xcode 打开项目：

```sh
open ChiHuoDaRen.xcodeproj
```

模拟器运行：

1. 在 Xcode 选择 `ChiHuoDaRen` scheme。
2. 选择任意 iPhone 模拟器。
3. 点击 Run。

iPhone 真机运行：

1. 用线连接 iPhone，解锁并信任这台 Mac。
2. 在 iPhone 开启开发者模式。
3. 在 Xcode 的 `Signing & Capabilities` 里选择自己的 Apple ID 开发团队。
4. 将运行目标切换为你的 iPhone。
5. 点击 Run。
